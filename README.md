## О проекте

SandBox Engine — это не просто игра, а **симуляция живой экосистемы**. Хищники и мирные существа рождаются, охотятся, размножаются, наследуют ДНК, мутируют и эволюционируют без скриптов. Всё поведение возникает из простых правил и генетики.

Игрок — **наблюдатель**. Никакого управления, только наблюдение за эмерджентными циклами Лотки-Вольтерры в реальном времени.

---

## 🔬 Возможности v0.3

- 🧬 **ДНК** — 6 генов (size, speed, attack, hp, courage, nutrition), наследование, мутации, запрет инцеста (family_id)
- 🐺 **Хищники** — охота, патруль с памятью мест, поиск партнёра, голод, рост
- 🐑 **Мирные** — кормление травой, бегство в логова, размножение, голод
- 🌿 **Растения** — рост, поедание, удобрение трупами, возрождение
- 🏠 **Логова** — убежища для мирных, хищники не входят
- 📊 **Полная параметризация** — любой параметр меняется через API
- 🧪 **Природные формулы** — метаболизм Клайбера, аллометрия, гауссовы мутации
- 🎥 **Камера** — панорамирование, зум, слежение за сущностью
- 📋 **Инфо-панель** — ПКМ по сущности показывает ДНК, голод, HP, возраст, family_id
