===========================================
SANDBOX (мир)
===========================================

SandBox(world_height=2000, world_width=2000, collision_enabled=True, show_borders=True)
  Создаёт окно Tkinter с холстом.
  
  Методы:
    spawn_plants(clusters=10, min_plants=10, max_plants=25, spread=100,
                 food_min=2, food_max=5, regrowth_min=0.003, regrowth_max=0.015,
                 bite_min=0.5, bite_max=1.5, bite_mult=0.8,
                 respawn_chance=0.0005, respawn_percent=0.3)
      Засевает траву кучками (clusters — количество кучек, 
      min_plants/max_plants — границы числа растений в кучке,
      spread — разброс от центра кучки в пикселях).
    
    spawn_random_entities(count, *types, color=None, **kwargs)
      Создаёт count случайных существ указанных типов (Peaceful, Predator, ...)
      в случайных местах мира. **kwargs прокидываются в __init__ существа.
    
    spawn_random_objects(count, color="#AFAEAE", form='rectangle')
      Создаёт count статичных объектов (камней) в случайных местах.
    
    spawn_den(x, y, size=40, color="#8B7355")
      Создаёт логово в указанной точке.
    
    create_player(color="#00FF00", speed=3)
      Создаёт игрока-наблюдателя.
    
    add_entity(entity)
      Добавляет существо в мир.
    
    add_object(obj)
      Добавляет статичный объект в мир.
    
    configure_entities(entity_type, breeding_enabled=None, **params)
      Массовая настройка: для всех существ указанного типа устанавливает
      переданные атрибуты. breeding_enabled=True/False — вкл/выкл размножение.

PEACEFUL (мирное травоядное)
===========================================

Peaceful(x, y, color="#8FBC8F",
  breed_chance=0.005,           # шанс размножения за кадр (при встрече пары)
  breeding_age=200,             # возраст зрелости в кадрах (60 кадров = 1 сек)
  child_size_percent=0.4,       # доля от max_size при рождении
  growth_divider=600,           # делитель для скорости роста (меньше = быстрее)
  start_hunger_min=50,          # мин значение голода при создании
  start_hunger_max=100,         # макс значение голода при создании
  
  # Скорости (множители dna.speed)
  flee_speed_close=0.9,         # бегство когда хищник ближе 50px
  flee_speed_mid=0.75,          # бегство когда хищник 50-100px
  flee_speed_far=0.6,           # бегство когда хищник дальше 100px
  graze_speed=0.3,              # движение к растению
  mate_speed=0.7,               # движение к партнёру
  wander_speed=0.5,             # блуждание без цели
  
  # Зрение
  vision_base=150,              # базовая дальность обнаружения хищника
  vision_per_size=5,            # бонус к дальности за размер хищника
  
  # Метаболизм
  base_metabolism=0.003,        # константа Клайбера (умножается на size^0.75)
  metabolism_per_size=0.004,    # устар. (используется степенная формула)
  
  # Стоимость действий (расход голода за кадр)
  flee_cost_close=0.08,         # бегство вблизи
  flee_cost_mid=0.05,           # бегство на средней дистанции
  flee_cost_far=0.03,           # бегство вдали
  mate_cost=0.02,               # поиск партнёра
  graze_cost=0.008,             # поиск еды
  wander_cost=0.005,            # блуждание
  
  # Размножение
  breed_hunger=70,              # минимальный голод для спаривания
  breed_hp_threshold=0.7,       # минимальная доля HP для спаривания (0.7 = 70%)
  breed_cooldown_base=200,      # базовый кулдаун после родов (кадры)
  breed_cooldown_per_size=20,   # добавочный кулдаун за единицу размера
  
  # Здоровье
  hp_regen_rate=0.05,           # восстановление HP за кадр (при сытости)
  hp_regen_threshold=80,        # порог голода для регенерации (> X = регенерирует)
  hp_damage_threshold=20,       # порог голода для урона (< X = теряет HP)
  hp_damage_rate=0.01           # урон HP за кадр (при голоде ниже порога)
)

===========================================
DNA (генетический код)
===========================================

DNA(size=None, speed=None, attack=None, hp=None,
    nutrition_value=None, starvation_threshold=None,
    family_id=None, courage=None)

  Все параметры опциональны.
  Если не указаны — вычисляются из size по природным формулам.
  
  Значения по умолчанию (при size=random(4, 10)):
    speed              = 1.5 + 0.5 * size^(1/3)
    nutrition_value    = 3 * size^(3/4)
    starvation_threshold = max(15, min(40, 10 + size * 2))
    courage            = min(0.9, 0.4 + size * 0.03)
    attack             = 1.5 + size * 0.4
    hp                 = 15 + size * 3
    family_id          = автоинкремент

  Методы:
    breed(other_dna) -> DNA
      Скрещивание: среднее родителей + гауссова мутация.
      Возвращает нового ребёнка с уникальным family_id.
      Инцест не проверяется здесь — проверка в find_mate().
